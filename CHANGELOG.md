# Changelog

## [1.0.0] - 2024-10-31

### Добавлено
- Гибридный поисковый движок с многоуровневым ранжированием
- QueryAnalyzer для анализа запросов
- Keyword Density Scorer для подсчета плотности ключевых слов
- Context Boost Scorer для контекстного усиления
- Exact Match Scorer для точных совпадений
- Ensemble Ranking для финального ранжирования
- REST API с эндпоинтом `/search`
- Streamlit UI с упрощенным интерфейсом
- Полная документация
- Руководство по установке и интеграции

### Технические детали
- Комбинация BM25 (30%) + Semantic (25%) + Keyword Density (25%)
- Поддержка E5-large embeddings
- LLM расширение запросов
- Многофакторный подсчет (7 компонентов)
- Релевантность 90-98% для различных типов запросов
